# Simplify.jobs — How to Use It

Simplify is not a job board. It's an application tracker and auto-fill tool that sits on top of other job boards and company career pages. The value is in the Chrome extension and the application tracking dashboard, not in finding new jobs.

---

## What Simplify actually does

**Auto-fill:** The Chrome extension detects when you're on a company's careers page and pre-fills application forms from your stored profile — name, email, experience, education, skills. This saves the 5–10 minutes of typing the same information into every Greenhouse or Workday form.

**Application tracker:** Every application you submit gets logged in Simplify's dashboard. You can see status (applied, viewed, interviewing, rejected, ghosted), add notes, and track the whole pipeline in one place. This is the part most people sleep on — knowing which applications are aging and which need a follow-up.

**Job feed:** Simplify aggregates listings from LinkedIn, Indeed, Glassdoor, and company career pages into one searchable feed. You can filter by role, tech stack, company size, and more. This is useful for discovery but not the primary reason to use it.

---

## How to set it up properly

1. Create an account at simplify.jobs
2. Fill in the profile completely — this is what auto-fill draws from. The more complete this is, the less manual correction you do on each application.
3. Install the Chrome extension
4. Store your most ATS-clean resume PDF in Simplify — this gets submitted to company ATS systems through the auto-fill

The profile in Simplify doesn't need to be perfectly crafted like LinkedIn or Naukri — it's just a data source for the auto-fill. But it does need to be accurate and complete.

---

## The resume you store in Simplify matters more than the profile

When applications route through a company ATS (Greenhouse, Lever, Workday), what gets submitted is your PDF resume, not your Simplify profile. This is the most important thing to understand about Simplify.

**Make sure the resume stored in Simplify is:**
- Single column, ATS-clean (see [README.md](README.md) for the ATS format checklist)
- Up-to-date — if you update your resume, re-upload it in Simplify too
- Tailored to the type of roles you're applying to (AI/SWE, not a generic one-size-fits-all version)

The auto-fill handles the form fields. The resume handles the actual screening. Don't let auto-fill speed make you lazy about resume quality.

---

## Using the application tracker properly

The tracker is only useful if you use it consistently. Every application — whether through Simplify's auto-fill, a direct company website, or LinkedIn Easy Apply — should be logged.

What to track for each application:
- Company and role title
- Date applied
- Where you found the listing
- Status (applied / phone screen scheduled / technical round / final / rejected / ghosted)
- Any notes (recruiter name, what they said, what to prepare)

This matters because:
- It prevents applying to the same role twice (more common than you'd expect when applying at volume)
- It shows you which companies are ghosting at the same stage so you can adjust your approach
- It gives you data on which platforms and which types of roles are actually converting to interviews

---

## ATS considerations when using Simplify

Auto-fill submits form fields quickly, but some company ATS systems have required fields that Simplify might skip or fill incorrectly. Always:
- Review the filled form before submitting — don't just click through
- Add a tailored cover letter or summary when the application asks for one (auto-fill doesn't handle open-text fields well)
- Double-check that the correct resume version was uploaded

Some companies also have anti-bot detection that may flag applications from the extension. If an application form seems to be rejecting your submission, try filling it manually.

---

## What Simplify is not good for

- Building recruiter relationships (it's a tool, not a network)
- Warm applications (for referrals or direct contacts, go through the company's own ATS directly)
- Startups with non-standard career pages (the auto-fill breaks on unusual form layouts)

For startups, Wellfound is the better tool. Simplify is best for applying at scale to mid-to-large companies that use standard ATS platforms.
