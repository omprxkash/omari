# Naukri Profile Summary

The profile summary is the first long-form text a recruiter reads on Naukri. It needs to be professional and specific — the current version ("I do AI/ML sometimes I love networking...") is too casual for a job application context.

---

## Current (old)

```
I do AI/ML sometimes I love networking with people so please don't hesitate to reach out :) 
My goal? trying to impact as many lives as I can with the use of AI.
```

This might work for a LinkedIn post, but in a recruiter search context it reads as someone who isn't taking the job search seriously. No numbers, no specifics, no indication of what you can actually do.

```
AI Engineer with 2+ years building production LLM systems — RAG pipelines, knowledge

Looking for AI Engineering and Software Engineering roles where I can build systems 
that actually reach users.
```

---

## Why this works

- **Opens with a credential phrase** ("2+ years building production LLM systems") rather than a personal statement — recruiters read this first and decide in one sentence whether to keep going
- **Specific domains listed** (RAG, knowledge graphs, agentic workflows) match recruiter keyword searches
- **Hard signals** (IEEE publications, patent) are named without over-explaining them
- **IIT research stints** — these carry weight on Naukri with Indian recruiters specifically
- **Tech stack** is concrete — not "AI/ML tools" but the actual libraries
- **Closes with intent** — "roles where I can build systems that actually reach users" is specific enough to be meaningful without being limiting

---

## Updating the summary over time

When you start a new role or finish a significant project, update the opening line to reflect the current experience level. The rest of the structure stays the same.
