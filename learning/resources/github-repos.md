# GitHub Repos Worth Bookmarking

Organized by what they're actually for.

## Build projects (not just tutorials)

**project-based-learning** (practical-tutorials)
https://github.com/practical-tutorials/project-based-learning
Tutorials for building real projects — grouped by language. Python, JS, Go, Rust, C++, and more. Find a project in a language you're learning and build it.

**build-your-own-x** (codecrafters-io)
https://github.com/codecrafters-io/build-your-own-x
Build Redis, Git, Docker, SQLite, a web server, a shell — from scratch. The most effective way to understand how things actually work.

**Web Dev Mini Projects** (Ayushparikh-code)
https://github.com/Ayushparikh-code/Web-dev-mini-projects
Small frontend projects for practising HTML, CSS, JS.

**100 AI / ML / DL Projects** (AdilShamim)
https://github.com/AdilShamim8/100-AI-Machine-Learning-Deep-Learnin-Projects
100 project implementations across ML, deep learning, NLP, and computer vision.

**DevOps Projects** (techiescamp)
https://github.com/techiescamp/devops-projects
Real DevOps projects — CI/CD, Kubernetes, monitoring, cloud infrastructure.

## Reference & Tools

**System Design Primer** (donnemartin)
https://github.com/donnemartin/system-design-primer

**The Book of Secret Knowledge** (trimstray)
https://github.com/trimstray/the-book-of-secret-knowledge
CLI tools, cheatsheets, hacks, manuals — for backend and DevOps engineers.

**Free Programming Books** (EbookFoundation)
https://github.com/EbookFoundation/free-programming-books
Free books on almost every language, framework, and topic.

**Free For Dev** (ripienaar)
https://github.com/ripienaar/free-for-dev
