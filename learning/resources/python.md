# Python

---

## Courses

**CS50P — Harvard (free)**
https://cs50.harvard.edu/python/
Harvard's intro to Python. Rigorous, project-based, free. One of the best structured beginner courses available.

**Python for Beginners – Full Course** (freeCodeCamp)
https://www.youtube.com/watch?v=eWRfhZUzrAc
Full beginner tutorial, no fluff.

**Python Full Course for Beginners** (Bro Code)
https://www.youtube.com/watch?v=ix9cRaBkVe0
Fast-paced, practical, covers all the fundamentals in one sitting.

**Python Tutorial for Beginners** (Programming with Mosh)
https://www.youtube.com/watch?v=K5KVEU3aaeQ
Clean, well-structured. Good for absolute beginners.

## Books

**Automate the Boring Stuff with Python** (free online)
https://automatetheboringstuff.com/
The best book for learning Python by doing real things — file manipulation, web scraping, Excel automation. Free online.

---

## Practice

**Practice Python**
https://www.practicepython.org/
Beginner exercises, one at a time, with solutions.

**Codewars Python Collection**
https://www.codewars.com/collections/python-practice-1
Kata-style problems. Good for building fluency once you know the basics.

**Python Tutor (visualizer)**
https://pythontutor.com/visualize.html#mode=edit
Paste code and watch it execute step by step. Invaluable for understanding how loops, recursion, and data structures actually work.

**fast.ai**

Practical-first deep learning in Python. Learn to train models before the math. Covers PyTorch. Free.
