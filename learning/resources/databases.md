## SQL

**SQL Tutorial for Beginners** (Programming with Mosh)
https://www.youtube.com/watch?v=7S_tz1z_5bA
The most-watched SQL tutorial. Clean, practical, covers everything you need for interviews and real projects.

**SQLBolt**
https://sqlbolt.com/
Interactive SQL lessons in the browser. Write real queries, see real results. Better than just watching.

## Tools

**DBHub** (bytebase)
https://github.com/bytebase/dbhub
Universal database explorer — connects to PostgreSQL, MySQL, SQLite, and more. Good for poking around databases without writing raw SQL.
