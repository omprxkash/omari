# Frontend & Design Resources

---

## Design Inspiration

**Dribbble**
https://dribbble.com/
The standard design portfolio platform. Browse for UI patterns, colour palettes, and layout ideas. Useful when you know what you want to build but need a visual reference.

**Godly**
https://godly.website/
Curated showcase of the best-designed websites. Study these for layout ideas, animation patterns, and how top studios handle typography and spacing.

**Mobbin**
https://mobbin.com/
Library of real mobile and web UI screenshots, organised by app and screen type. Good for studying how specific UX patterns (onboarding, empty states, settings) are handled in production apps.

**Awwwards — CSS & JS Animations**
https://www.awwwards.com/awwwards/collections/css-js-animations/
Award-winning animation examples. Go here when you need animation inspiration that's both technically impressive and tasteful.

## Hosting

**Railway**
https://railway.com/
The easiest way to deploy full-stack apps. Connect a GitHub repo, set env vars, and it's live. No config files needed for most frameworks. Free tier is generous for side projects.

## Dev tools

**View Page Source**
https://www.view-page-source.com/
Paste any URL and read its HTML source cleanly. Useful for inspecting how a site is structured without opening DevTools.

**iLoveIMG — Resize Image**
https://www.iloveimg.com/resize-image
Quick browser-based image resizing. No signup needed.

**Full-size screenshot (browser)**
Open DevTools → `Cmd+Shift+P` (Mac) / `Ctrl+Shift+P` (Windows) → type "screenshot" → select "Capture full size screenshot". Takes a full-page screenshot without any extension.

**ffmpeg**
https://ffmpeg.org/
CLI tool for video and audio processing — convert formats, cut clips, extract audio, compress video. Steep learning curve but nothing else comes close for scripted media processing.
