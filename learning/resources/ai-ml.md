## LLMs & Transformers

**Create a Large Language Model from Scratch with Python** (freeCodeCamp)
https://www.youtube.com/watch?v=UU1WVnMk4E8
Build an LLM from the ground up. Good for understanding what's actually happening inside.

https://github.com/HandsOnLLM/Hands-On-Large-Language-Models
Official code for the O'Reilly "Hands-On LLMs" book. Covers tokenization, fine-tuning, embeddings, inference.

https://github.com/huggingface/transformers
The main library for working with pre-trained models. Read the docs and examples directly.

**HuggingFace Agents Course**
https://huggingface.co/learn/agents-course/

**LLMs in Production vs Development** (Stripe Developers)
https://www.youtube.com/watch?v=MB3XZG4jcQA
19-minute breakdown of the practical gaps between building with LLMs locally vs. at production scale. Good orientation before you start deploying anything.
