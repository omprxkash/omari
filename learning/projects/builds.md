# Things to Build

Reading and watching only gets you so far. Projects are where things click. These are organised by what you are trying to practice, not by difficulty — start where your current skills are.

---

## Python Fundamentals

Small scripts that make something real:

- **File organiser** — scan a folder, sort files into subfolders by extension (images, docs, videos)
- **Web scraper** — scrape job listings or product prices from a site using BeautifulSoup
- **CLI flashcard app** — read questions from a text file, quiz yourself, track score
- **Expense tracker** — read/write CSV, filter by category, print monthly summaries
- **YouTube transcript downloader** — use `yt-dlp` to pull transcripts from a playlist

---

## Data Structures & Algorithms Practice

Build the data structure from scratch, then use it:

- **Implement a stack, queue, and linked list** in plain Python (no libraries)
- **Build a tiny search engine** — tokenise a set of text files, build an inverted index, return matches for a query
- **Graph pathfinder** — read a list of cities and distances, find shortest path with BFS/Dijkstra
- **LRU Cache** — implement using a hash map + doubly linked list (classic interview problem)

---

## APIs

- **Weather dashboard** — call a free weather API, display data cleanly in the terminal or a simple HTML page
- **Stack Overflow question finder** — call the Stack Exchange API, filter by tag, show unanswered questions
- **GitHub profile analyser** — use the GitHub API to show a user's most active repos and languages
- **Currency converter** — call an exchange rate API, convert between currencies from the CLI

---

## AI / LLM

- **PDF chatbot** — upload a PDF, chunk it, embed it, answer questions from it using RAG
- **Personal knowledge base** — index your own notes into a vector DB, query them with natural language
- **Summariser API** — FastAPI endpoint that accepts a URL, fetches the page, summarises with an LLM
- **Resume screener** — embed a job description and a set of CVs, rank by semantic similarity
- **Agentic research tool** — give an LLM access to a search tool, let it research a topic and write a report

---

## Full Stack

- **Link shortener** — React frontend, FastAPI backend, SQLite database, deploy on Railway
- **To-do app with auth** — user accounts, login, per-user task lists — the classic but do it properly
- **SaaS landing page + waitlist** — static site with an email capture form that writes to a database
- **Full stack AI chatbot** — React frontend, FastAPI backend, streaming LLM responses, conversation history
